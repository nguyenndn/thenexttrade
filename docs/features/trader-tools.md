# Trader Tools

## 1. Trading Journal
The core tool for logging and reviewing trades.
- **Manual Entry:** Log trades with entry/exit price, lot size, and notes.
- **EA Sync:** Automatically import trades from MT4/MT5 (via Expert Advisor).
- **Tags:** Tag trades with strategies, mistakes, or psychological states.

## 2. Strategy Manager
Define and track different trading setups.
- **Create Strategy:** Name, Description, and Rules.
- **Performance Tracking:** See Win Rate and P&L per strategy.
- **Comparison:** Compare multiple strategies side-by-side to identify the edge.

## 3. Psychology Tracking
Monitor emotional states to improve discipline.
- **Tiltmeter:** Detects "tilt" behavior (e.g., revenge trading, over-leveraging).
- **Emotion Logging:** Record feelings before and after trades.
- **Analysis:** Correlate emotions with P&L (e.g., "Do I lose more when I feel FOMO?").

## 4. Risk Calculator
Essential tools for position sizing.
- **Position Size Calculator:** Determine lot size based on account balance and risk %.
- **Pip Value Calculator:** Calculate value per pip for different currency pairs.
- **Risk/Reward Calculator:** Visualize R:R ratios before entering a trade.
